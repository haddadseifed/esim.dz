# eSIM DZ — Minimal Professional Bundle

This is a **lightweight, production-ready** static site bundle for **eSIM DZ Algeria**.  
It contains only the essentials to launch quickly on **GitHub Pages** with a custom domain.

## What’s inside
- `index.html` — modern landing page (hero, about, plans, how, FAQ, contact)
- `style.css` — minimal responsive design (Apple/Google-like)
- `script.js` — demo checkout (modal), order success with QR preview & PDF
- `CNAME` — set to `esimdz.dpdns.org` (required for custom domain on GitHub Pages)
- `/assets` — logo, favicon, and images

> **Note:** All payments are simulated client-side for demo purposes only.

## Deploy to GitHub Pages
1. Create a public repo named **`eSIM-DZ`** under your GitHub account `haddadseifed`.
2. Upload all files and folders (root, not inside another folder).
3. Go to **Settings → Pages**:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` / root (`/`)
4. Add your custom domain: `esimdz.dpdns.org` (CNAME file is already included).
5. Wait a few minutes until HTTPS is active.

## Edit content
- Plans and prices: edit the three plan cards in `index.html` (search for “Popular Plans”).
- Colors and fonts: see `style.css` (`--primary`, `--accent`) and Google Fonts.
- Logo/Favicon: replace `assets/logo.png` and `assets/favicon.png` with your files.
